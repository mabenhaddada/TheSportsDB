# TheSportsDB

A Swift project that integrates with TheSportsDB API.

## Prerequisites

- Xcode (26.0)
- Swift toolchain that matches the projectâ€™s settings
- A TheSportsDB API key (free tier available: 123)


## Clone repository

```bash
$ git clone git@github.com:mabenhaddada/TheSportsDB.git
$ cd TheSportsDB
```

## Install dependencies

```bash
$ make deps
```


## Setup TheSportsDB API key in App.debug.xcconfig file

```bash
THESPORTSDB_API_KEY=123
```
